# Data Professional Survey Analysis Dashboard

## 📊 Project Overview
This project presents an interactive Power BI dashboard built to analyze global data professional survey responses. The dashboard provides insights into salary trends, job roles, programming language preferences, and career satisfaction metrics.

## 🎯 Objective
To analyze workforce trends in the data industry and extract actionable insights using data modeling and visualization techniques.

## 📁 Dataset
- 630+ survey responses
- Includes job titles, country, salary, programming language, work-life balance, and career satisfaction metrics

## 🛠 Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Calculated Measures & KPIs)
- Data Modeling

## 📈 Key KPIs Created
- Average Salary
- Count of Survey Takers
- Work-Life Balance Score
- Salary Satisfaction Score
- Career Entry Difficulty Distribution

## 📊 Dashboard Features
- Salary analysis by Job Title
- Country-wise survey distribution
- Programming language preference trends
- Interactive slicers and drill-down reports
- Dynamic KPI cards using DAX

## 🔍 Key Insights
- Data Scientists show the highest average salary range.
- Python is the most preferred programming language.
- Majority of respondents report moderate difficulty entering the data field.
- Work-life balance scores vary significantly across job roles.
